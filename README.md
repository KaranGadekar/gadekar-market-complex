# Gadekar Market Complex (Static Site)

Single-page static website (HTML + CSS + minimal JavaScript) for **Gadekar Market Complex**.

## What’s included
- `index.html` — semantic single-page layout
- `styles.css` — modern, responsive styling (no frameworks)
- `script.js` — minimal JS (mobile nav toggle + footer year)

## Run locally (optional)
You can open `index.html` directly in a browser, or run a tiny local server.

### PowerShell (Python)
```powershell
cd "c:\Users\karan\Desktop\Gadekar plaza"
python -m http.server 5173
```
Then open `http://localhost:5173`.

## Deploy
Upload the folder to Netlify / Vercel as a static site (no build step needed).
